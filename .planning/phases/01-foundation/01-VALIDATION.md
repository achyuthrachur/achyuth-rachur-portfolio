# Phase 1 — Validation Strategy

> Per-phase validation contract for feedback sampling during execution.

---

## Test Infrastructure

| Property | Value |
|----------|-------|
| **Framework** | TypeScript compiler (`tsc --noEmit`) + Next.js build |
| **Config file** | `tsconfig.json` (generated by create-next-app) |
| **Quick run command** | `npm run typecheck` (alias for `tsc --noEmit`) |
| **Full suite command** | `npm run build` |
| **Estimated runtime** | ~15 seconds (typecheck), ~45 seconds (build) |

> **Note:** Phase 1 is a scaffold phase — no application logic exists yet to unit test. Validation is build-time (TypeScript compilation) and structural (file existence checks). Vitest will be installed in a later phase when there is application logic to test.

---

## Sampling Rate

- **After every task commit:** Run `npm run typecheck`
- **After every plan wave:** Run `npm run build`
- **Before `/gsd:verify-work`:** Full suite (`npm run build`) must succeed and produce `/out`
- **Max feedback latency:** ~60 seconds
