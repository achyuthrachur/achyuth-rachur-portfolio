  <action>
Install all three animation/icon packages. These are the SCAF-05 dependencies.

Step 1 — Install animejs and motion together:

```bash
npm install animejs motion
```

Both packages have proper React 19 peer dependency declarations and should install without conflicts.

Do NOT install `@types/animejs` — animejs v4 ships built-in TypeScript definitions. Installing @types/animejs would cover the v3 API and cause type conflicts.

Do NOT install `framer-motion` — it is the old package name. The `motion` package is the correct 2026 package.

Step 2 — Install iconsax-react with legacy peer deps flag (required because iconsax-react@0.0.8 declares react@^17||^18 peer deps, but the project uses React 19):

```bash
npm install iconsax-react --legacy-peer-deps
```

Step 3 — Verify all three packages are importable:

```bash
node -e "
  const animejs = require('animejs');
  const motion = require('motion');
  const iconsax = require('iconsax-react');
  console.log('animejs:', typeof animejs);
  console.log('motion:', typeof motion);
  console.log('iconsax:', typeof iconsax);
  console.log('ALL DEPS OK');
"
```

Expected output: Lines showing type of each module and "ALL DEPS OK". If any require() fails, check the error and re-install that package.

Step 4 — Run TypeScript check to confirm no type conflicts:

```bash
npm run typecheck
```

If typecheck fails with animejs type errors (unlikely but possible), the fix is to NOT have @types/animejs installed — check `npm list @types/animejs` and remove if present: `npm uninstall @types/animejs`.

8. Verify the file has a named export: `export function [ComponentName](...)`

POST-INSTALL REQUIREMENTS FOR EACH COMPONENT:

After installing/copying each component, verify these invariants:
1. The file MUST have `'use client';` as the first line (React Bits components use browser APIs)
2. The file MUST use `import { ... } from 'motion/react'` — NOT `from 'framer-motion'`
3. The file MUST have a named export (not default export): `export function ComponentName` or `export const ComponentName`
