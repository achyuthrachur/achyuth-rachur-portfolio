**Anime.js bounce effect on scroll indicator (HERO-06) — with prefers-reduced-motion guard:**
```typescript
useEffect(() => {
  const prefersReduced = window.matchMedia('(prefers-reduced-motion: reduce)').matches;
  if (prefersReduced || !scrollIndicatorRef.current) return;

  const anim = animate(scrollIndicatorRef.current, {
    translateY: [0, 8, 0],
    duration: 1200,
    loop: true,
    ease: 'inOutSine',
  });

  return () => anim.cancel(); // cleanup required — prevents RAF leak on hot reload
}, []);
```
