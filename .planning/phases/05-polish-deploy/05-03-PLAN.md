  </what-built>
  <how-to-verify>
**Cloudflare Pages connection — step-by-step walkthrough:**

1. Go to https://dash.cloudflare.com and sign in to your existing account

2. In the left sidebar, click **"Workers & Pages"**

3. Click **"Pages"** tab at the top

4. Click **"Create a project"** (blue button, top right area)

5. Click **"Connect to Git"** (not "Direct Upload")

6. If GitHub is not already authorized:
   - Click "Add account" or "Authorize Cloudflare Pages"
   - GitHub will ask you to authorize — click "Authorize Cloudflare Pages"
   - You may be asked to install the Cloudflare GitHub App — install it for "Only select repositories" and select `achyuth-rachur-portfolio`

7. You should now see a list of your GitHub repositories. Find and click **"achyuth-rachur-portfolio"** → click **"Begin setup"**

8. Configure build settings on the next screen:
   - **Project name:** `achyuth-rachur-portfolio` (auto-filled — leave as-is)
   - **Production branch:** `master`
   - **Framework preset:** `None` (do NOT select Next.js — we use static export, not Cloudflare's Next.js adapter)
   - **Build command:** `npm run build`
   - **Build output directory:** `out` (IMPORTANT — not `dist` or `.next`)
   - **Environment variables:** leave empty

9. Click **"Save and Deploy"**

10. Cloudflare will start a deployment. You can watch the build log in real time. It typically takes 2-4 minutes.
    - Build log should show: "Installing dependencies", "Running npm run build", "Uploading files"
    - Build should succeed (green checkmark)

11. After deployment succeeds, Cloudflare shows your project page with a URL like:
    `https://achyuth-rachur-portfolio.pages.dev`
    (or a hash-based URL like `https://abc123.achyuth-rachur-portfolio.pages.dev`)

12. Click the URL and verify:
    - Page loads with the Crowe Indigo Dark hero background
    - "Achyuth Rachur" name appears (SplitText animation)
    - Sticky nav is visible at top
    - Scrolling through all sections works
    - Contact section loads
    - Footer visible at bottom

- If you see "Error: Cannot find module" errors, the Node.js version in Cloudflare may differ. Add environment variable: `NODE_VERSION` = `20`
